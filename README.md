

**Neighborhood Fix Map+** is a static, map-based civic awareness web project designed to visually represent common neighbourhood issues such as potholes, broken streetlights, garbage dumping, unsafe animal zones, and crime-sensitive areas.  
The project focuses on **improving civic awareness, safety, and accessibility** through a clean and intuitive interface built using **HTML and CSS**.

Urban neighbourhoods often face recurring civic problems that remain unreported due to the lack of a centralized and user-friendly platform. Neighborhood Fix Map+ addresses this gap by offering a **visual, color-coded map-based solution** that helps citizens identify problem areas and understand neighbourhood risks at a glance.

This project is developed as part of an **Activity Based Assessment**, following the **Design Thinking methodology**.
